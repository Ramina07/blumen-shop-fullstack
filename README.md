# Blumen Boutique — Fullstack Web Application

Полнофункциональное веб-приложение для автоматизации заказов в цветочном магазине. Проект включает в себя интерактивную витрину, серверную логику на Node.js и интеграцию с Telegram для мгновенного уведомления администратора о новых заказах.

**Live Demo:** [https://blumen-boutique.onrender.com](https://blumen-boutique.onrender.com)

---

## Технологический стек

### Frontend
* **HTML5 и CSS3** — адаптивная верстка и современный дизайн.
* **JavaScript (ES6+)** — динамическое управление корзиной и асинхронные запросы.
* **Fetch API** — взаимодействие с серверной частью без перезагрузки страниц.

### Backend
* **Node.js** — программная платформа для серверной части.
* **Express.js** — веб-фреймворк для обработки маршрутов и API-запросов.
* **MongoDB и Mongoose** — база данных NoSQL для хранения заказов и каталога.
* **Telegram Bot API** — система мгновенных уведомлений для владельца бизнеса.

### DevOps и Tools
* **Git и GitHub** — контроль версий.
* **Render** — деплоймент и хостинг приложения.
* **Dotenv** — безопасное управление переменными окружения.

---

## Основной функционал

* **Динамический каталог:** Отображение товаров из облачной базы данных.
* **Система корзины:** Добавление товаров и расчет итоговой стоимости на стороне клиента.
* **Клиент-серверное взаимодействие:** Обработка форм заказов через кастомные API-эндпоинты.
* **Интеграция с Telegram:** Автоматическая отправка деталей заказа (имя, телефон, состав заказа) в чат администратора.
* **Database Cloud Sync:** Синхронизация данных с MongoDB Atlas.

---

## Архитектура проекта

Проект реализован в рамках клиент-серверной архитектуры:
- `backend/` — логика сервера, модели данных и маршруты API.
- `frontend/` — статические файлы (UI/UX часть).
- `server.js` — точка входа, объединяющая раздачу статики и работу API.
